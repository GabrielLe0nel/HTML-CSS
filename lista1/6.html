Como podemos incluir CSS em documentos HTML? Cite e explique as diferentes formas de incorporar CSS (inline, internal, external) e comente o impacto de cada método no desempenho e na organização do código.

        
    
            
  1. CSS Inline: Como funciona: O CSS é aplicado diretamente em um elemento HTML usando o atributo <'style'>.
    2. 
            
  2. CSS Internal: Como funciona: O CSS é incluído dentro da tag <'style'> no <'head'> do documento HTML.
    3. 
            
  3. CSS External:Como funciona: O CSS é colocado em um arquivo separado com a extensão .css, que é vinculado ao documento HTML através da tag <'link'> no <'head'>.
