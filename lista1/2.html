O que são seletores CSS? Quais são os tipos existentes?

Em CSS, os seletores são usados para estilizar as páginas web, direcionando os elementos HTML.

Tipos existentes e exemplos;

  • Seletores de tipo. Exemplo: h1 { color: blue; }
  • Seletores de classe. Todos os elementos da classe box serão selecionados. Exemplo: .box { margin: 10px; }
  • Seletores de ID. Seleciona um único elemento com o ID "padrao". Exemplo: #padrao { font-size: 20px; }
  • Seletores de atributos. Seleciona elementos com um atributo específico, como type="text". Exemplo: input[type="text"] { border: 1px solid gray; }
  • Seletores de Pseudo classes. Seleciona elementos em um estado específico, como quando o mouse está sobre eles. Exemplo: a:hover { text-decoration: underline; }
  • Seletores de pseudo-elementos. Seleciona uma parte específica de um elemento, como antes do seu conteúdo. Exemplo: p::before { content: "Nota: "; color: red; }
  • Seletores de Combinadores. Seleciona elementos com base na relação entre eles, como um "<'p'>"" que é filho direto de uma "<'div'>"".Exemplo: div > p { font-weight: bold; }
