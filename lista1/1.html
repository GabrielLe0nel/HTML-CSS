O que é uma regra CSS? Explique o conceito de regra CSS e forneça um exemplo prático que inclua um seletor, uma propriedade e um valor.

Regra CSS pode ser entendido como um conjunto de regras utilizadas na linguagem CSS que define como será aplicado estilo a um ou mais elementos HTML.


Formato geral de uma regra CSS:

        <style>
            /* Seletor: h1; Propriedade: color; Valor: darkblue */
            h1 {
                color: darkblue; /* O texto do h1 ficará azul escuro */
            }
        </style>
